The n8n Logic Pipeline

A transparent look into our exact 5-step automation routing.

1. Webhook Payload

Captures Target URL, City, and Client Credentials instantly upon submission.

2. Site Scraper Node

HTTP Request executes a GET command to extract the raw HTML structure of the client's business.

3. Gemini Initial Context

LLM analyzes the scraped data to perfectly define the client's niche (e.g., 'Modular Kitchens' vs 'Full Interior').

4. Serper API Call

Queries Google's live database for the exact niche + target city to pull the Top 3 organically ranking competitors.

5. Strategy Synthesis & Delivery

Final Gemini Node correlates all data, writes the Masterplan in HTML, and routes it directly to the Gmail Node for dispatch.
